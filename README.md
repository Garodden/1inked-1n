### 1. 💻 개발 환경
- 프론트엔드: React, HTML, CSS, JavaScript, tailwind, shadcn-ui
- 백엔드: Spring Boot, Hibernate,
- DB: MySQL
- 배포 및 클라우드: AWS EC2, AWS S3, AWS RDS
- CI/CD: GitHub Actions, AWS CodePlay
- 협업도구:Github, Notion, Miro, ERD Cloud, Figma
